# Setup & Development

This document contains additional setup notes for the Stock Management project.

1. Install Node.js (LTS recommended) and npm.
2. From project root run `npm install`.
3. Start dev server with `npm run dev` and open `http://localhost:5173`.

Troubleshooting:
- If styles don't update, try a hard refresh (Ctrl+F5) or restart the dev server.
- If Git pushes fail, ensure your Git credentials are configured (SSH key or Git credential manager).
